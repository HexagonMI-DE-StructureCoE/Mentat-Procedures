# Mentat-Procedures

This repository contains a collection of MSC Marc Mentat plugins.

## How to install
1. Download the project
2. In Marc/Mentat GUI activate User Plugins
3. Select the folder that contains the plugins 
4. Restart MSC Marc Mentat
