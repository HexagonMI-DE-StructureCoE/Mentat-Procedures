Analysis Warnings & Errors

This plugin uses a Perl script to scan the Marc output file of the current job and extracts information from the warning and error messages that appear in this file. The plugin creates one or more sets to display the following information:

  • List of nodes that are separating

  • List of INSERT nodes that have not converged

  • List of contact node sliding along segment

  • List of contact nodes belonging to more than 1 body

  • List of contact nodes that have a DOF conflict

  • List of inside out elements

  • List of nodes that have max displacement change and increment from convergence criteria

  • List of nodes that have max residual change and increment from convergence criteria

  • Contacting Nodes

  • Bad beam section

  • Bad quadratic projection

  • IPC check (ddu multiplied by x to avoid large displacements...)

  • IPC check (ddu multiplied by x to avoid penetration)

  • too many nodes joined to node

  • Projection for contact node failed

  • Bad rigid body orientation

  • Node separated 5 times and will be skipped

  • Incorrect degenerated Hex elements

  • Start of assembly

  • Start of matrix solution

  • End of matrix solution

  • Start of global remeshng

  • Axisymmetric element has negative radius

  • IPC check (ddu multiplied by x to avoid penetration) - but only 1e-6 values

  • Zero length in element

  • Tying debug capture

This plugin requires that Perl is installed on the machine.
